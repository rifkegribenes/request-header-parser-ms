## Request Header Parser Microservice (Freecodecamp Project)

### User stories:

* I can get the IP address, language and operating system for my browser.

#### Example Output:

```javascript
	{
	"ipaddress":"0.0.0.0",
	"language":"en-US",
	"software":"Macintosh; Intel Mac OS X 10_12_6"
	}

```
